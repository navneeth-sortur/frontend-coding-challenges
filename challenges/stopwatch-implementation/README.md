# Stopwatch - Frontend Machine Coding Round

A simple stopwatch application built as part of a frontend machine coding round.  
The app supports **Start**, **Stop**, **Reset**, and shows a smoothly updating live timer.

## Problem Statement

Implement a functional stopwatch in React with the following features:

- Start the timer
- Stop the timer
- Reset the timer
- Display time in `MM:SS:MS` format
- Timer should update smoothly without flicker

## Tech Stack

- **React**
- **TypeScript**
- **TailwindCSS** for basic styling
