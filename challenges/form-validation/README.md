# React Form Validation Challenge

## Problem Statement

Build a React application that validates a form containing the following input fields:

1. **Name**
2. **Email**
3. **Phone Number**
4. **Blog URL**

A **"Verify"** button should validate the form and display a message below the form indicating whether the input is valid or contains errors.

---

## Validation Rules

### Name

- Must be between **3 and 30 characters**
- Only **uppercase and lowercase English alphabets** allowed

### Email

- Must be a valid email address format

### Phone Number

- Must be a **10-digit number**
- Must **not start with 0 or 1**

### Blog URL

- Must be a valid standard URL

---

## Requirements

- Use **React functional components**
- Follow **clean component-based architecture**
- Maintain **clean and readable state management**
- Display validation feedback clearly

---

## Outcome

- Displays error messages for invalid input
- Displays a success message when all inputs are valid
